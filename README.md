# URL Shortener
ASP.NET Core and Blazor WebAssembly Url Shortener.

<img src="./Docs/image.png" />

## Quick Start
```sh
# Start docker containers
docker compose up -d --build

# Run the server
cd UrlShortener/Backend
dotnet run

# Run the client
cd UrlShortener/Frontend
dotnet run
```

Website URL: http://localhost:5265/.
